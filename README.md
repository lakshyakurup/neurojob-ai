# 🚀 NeuroJob AI

> Stop applying blindly. Start applying smart.

NeuroJob AI analyzes any job description and tells you exactly how well your resume matches it, generates a tailored cover letter in seconds, finds missing ATS keywords, and predicts interview questions — all powered by Gemini AI.

## ✨ Features

- ✅ Resume fit score with detailed breakdown
- ✅ One-click AI cover letter generator
- ✅ ATS keyword gap analyzer
- ✅ Interview question predictor
- ✅ Clean minimal UI, fully responsive

## 🛠️ Tech Stack

- **Frontend:** Next.js 14, TailwindCSS, shadcn/ui
- **AI:** Google Gemini API
- **Auth:** Clerk
- **Database:** Supabase
- **Deployment:** Vercel

## 🚀 Getting Started

```bash
git clone https://github.com/lakshyakurup/neurojob-ai
cd neurojob-ai
npm install
npm run dev
```
